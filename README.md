## Learning Philosophy

I believe the best bioinformatics work sits at the intersection of three things:

1. **Deep biological intuition** — understanding *why* the data behaves the way it does
2. **Solid computational skills** — being able to build and run the analysis yourself
3. **Clear communication** — turning results into biological insight, not just numbers

Every project in this profile will aim to demonstrate all three.
